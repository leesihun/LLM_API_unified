## Task
{instruction}
{context_block}

Write one complete executable Python script that performs the task end-to-end.

Requirements:
- Include all imports at the top.
- Use relative paths inside the scratch workspace unless the task gives an
  absolute path.
- Read and write text with `encoding="utf-8"`.
- Print the result, output file path, or a clear success message.
- Validate external inputs (file existence, parse errors, network, subprocess)
  and on failure print a concrete error to stderr and `sys.exit(1)`.
- No `input()`, no GUI prompts, no `plt.show()`, no blocking displays.
- Do not print secrets, tokens, or credential file contents.
- Output exactly one fenced Python code block, OR the line
  `NEEDS_CLARIFICATION: ...` if one missing detail blocks a safe script.
